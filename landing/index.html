About the Program

May ongoing beauty quality control program in partnership with selected retail stores like Watsons, SM Beauty/LOOK, and Beauty Bar. Selected participants receive: Madagascar Centella Travel Kit — FREE kapalit ng honest video feedback.

Madagascar Centella Travel Kit

  • Centella Unscented Serum — calming & barrier repair
  • Centella Unscented Toner — hydration prep
  • Centella Unscented Cream — intensive moisture
  • Centella Safe Sun SPF50+ — daily protection

Formulated without fragrance, alcohol, or harsh irritants. Designed for sensitive skin.

Paano gumagana ang program

  1. Download Quality Connect App
  2. Select available product set
  3. Generate personal one-time QR code
  4. Pick up from partner store
  5. Submit honest video review after use

Android Installation Guide

App is not available on Google Play because it is a private feedback program. Normal Android message about "unknown sources" may appear — this is standard security behaviour.

  • Download starts automatically
  • Open downloaded file (app.apk)
  • Allow one-time install permission
  • Install and open app

Program Rules

  • 1 participant = 1 travel kit
  • QR code is one-time use only
  • No payment or purchase required
  • No subscription or credit card
  • Honest feedback required (positive or negative)
